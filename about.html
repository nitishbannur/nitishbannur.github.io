About Me!

Hello everybody! I'm Nitish, I graduated in Computer Engineering in 2019 from MIT College of Engineering, Pune.

I work as a BI Developer on offering Analyic BI solutions, which include managing data sources, extracting them and loading them into Reporting tools to create insightful dashboard visualizations.

A typical work-day for me revolves around:

  • Assisting with the development of the data model(s) for the data warehouse.
  • Perform ETL operations to further process the data to pipeline in Reporting.
  • Develop Reports and Dashboards using Tableau.
  • Automate any process such as refreshing the dashboard or running queries daily so that manual intervention is as minimal as possible.
  • Work in an agile environment and contribute to the improvement of our development processes.
  • Unit test reports and dashboards for UI issues, Data error and fix bugs related to them.
